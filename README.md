# Data Analysis Project 

This portfolio presents data analysis projects built on real-world datasets from an Italian company (Foorban) delivering smart fridges, coffee machines, and water supply services.

The projects focus on logistics performance, operational efficiency, customer segmentation, and data-driven business insights.

## Skills
- Python (pandas, numpy, matplotlib, seaborn)
- SQL (joins, aggregations, data extraction)
- Excel (data cleaning, pivot tables)
- Power BI (dashboards and KPIs)
- Exploratory Data Analysis (EDA)
- Data storytelling and reporting

- Customer Segmentation and Clustering
- Operational Dashboard (Power BI)
